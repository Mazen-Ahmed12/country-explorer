# 🌍 Country Explorer

A responsive web app to explore information about countries around the world.

## Features

- 🔍 Search countries by name
- 🌎 Filter by region
- 📊 View detailed country information
- ❤️ Save favourite countries
- 🌙 Dark mode support

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- ShadCN UI
- Zustand
- React Query (TanStack Query)

## Getting Started

``` bash

npm install
npm run dev

```

Open [http://localhost:3000](http://localhost:3000)
